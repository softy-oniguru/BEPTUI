## âœ… Key Implementation Details

### 1. **Proper Imports in Template**

All template files import from the `bertui` package:

```jsx
// âœ… Correct - Users import from package
;
import { Link } from 'bertui/router';

// âŒ Wrong - Don't use relative imports
import { Link } from '../.bertui/router';
```

### 2. **Router Export Path**

The `bertui` package exports router utilities:

```json
// package.json
{
  "exports": {
    "./router": "./src/router/client-exports.js"
  }
}
```

### 3. **Generated Router Uses Package Imports**

The auto-generated `.bertui/router.js` imports pages correctly:

```javascript
// Generated by compiler
import Page0 from '../src/pages/index.jsx';
import Page1 from '../src/pages/about.jsx';
```

### 4. **Dev Server Handles Routing**

The dev server serves all routes as SPA:

```javascript
// Any route returns the same HTML
app.get('/*', async () => {
  // Serve index.html with router
});
```
